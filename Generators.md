## ⭐ Что такое итератор и итерируемый объект?
- Итерируемый объект (Iterable) — это объект, по которому можно итерироваться (например, в цикле for).
- Чтобы объект был итерируемым, он должен реализовывать метод __iter__(), который возвращает итератор.
Итератор должен реализовывать:

__iter__() — возвращает самого себя

__next__() — возвращает следующий элемент или бросает StopIteration

Важное отличие:
- Итерируемый объект — "коллекция" (может создать итератор)
- Итератор — "указатель" на позицию в коллекции (одноразовый)

## ⭐ Функция-генератор, объект-генератор, генераторное выражение
Функция-генератор — это функция, которая содержит хотя бы один yield.

Объект-генератор — это то, что возвращается при вызове функции-генератора.

```Python
gen = simple_gen()   # gen — это генератор (объект-генератор)
```

### Чем генератор отличается от итератора?

Генератор — это особый вид итератора, который:

Приостанавливает выполнение функции при yield
Сохраняет локальные переменные и состояние стека
Может возобновить работу позже

### Что такое подгенераторы и yield from?
yield from (с Python 3.3) позволяет делегировать генерацию значений другому генератору или итерируемому объекту.

```Python
def subgen():
    yield 1
    yield 2
    yield 3

def main_gen():
    yield 0
    yield from subgen()      # ← магия
    yield 4
```

## Контейнерные и плоские последовательности

Контейнерные последовательности (Container sequences)
Хранят ссылки на объекты.

- list
- tuple
- collections.deque
- collections.abc.Sequence (в общем случае)

Особенности:

- Могут хранить объекты разных типов.
- Каждый элемент — это указатель на объект (8 байт на ссылку).
- Больший overhead памяти.
- Гибкие.

Плоские последовательности (Flat sequences)
Хранят значения напрямую в непрерывном блоке памяти (как массив в C).

- str
- bytes
- bytearray
- array.array
- memoryview
- numpy.ndarray (в какой-то степени)

Особенности:

Все элементы одного типа (int, float, char и т.д.).
Значительно экономнее по памяти.
Быстрее в обработке.
Менее гибкие (нельзя хранить разные типы).

```Python
import sys
import array

a = [1, 2, 3, 4, 5]                    # контейнерная
b = array.array('i', [1, 2, 3, 4, 5]) # плоская (signed int)

print(sys.getsizeof(a))   # ~120 байт + объекты
print(sys.getsizeof(b))   # ~68 байт (гораздо меньше)
```

Контейнерные — для удобства и гибкости.
Плоские — для экономии памяти и производительности при больших объёмах однотипных данных.
